🔧 安装

系统要求

  • Java 8 及以上，Spring 5.0 及以上
  • MySQL 8 及以上，当前配置只支持 MySQL 数据库，但 SQL 执行的数据不限 MySQL
  • 轻量级设计，对系统要求低，单核/512MB RAM 即可运行服务

下载与源码

数据服务依赖于 AJ 框架，严格来说它是该框架下的一个组件，故引入 AJ 相关依赖即可。AJ 框架足够轻量级，包括依赖包在内的 JAR 包约 200 多kb。
